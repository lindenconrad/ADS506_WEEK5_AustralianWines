# ADS506 WEEK5 Australian Wines Forecasting App 

This repository contains the source code and dataset for an interactive Shiny application that explores and forecasts Australian wine production using modern time series methods. The app allows users to filter data by varietal and date range, configure forecasting models, and compare performance across ETS and ARIMA models.

## Contains
- app.R — Main Shiny application script
- AustralianWines.csv — Monthly Australian wine sales data
- README.md — Project overview and usage instructions

The deployed Shiny app is available at:
https://lconrad-usd.shinyapps.io/week5hw_australianwines/
